# Changelog

All notable changes to Phast Framework will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Initial release of Phast Framework
- PSR-7, PSR-11, PSR-15, PSR-3, PSR-6, PSR-16, and PSR-20 compliance
- Dependency injection container with automatic resolution
- Routing system with parameter binding
- Middleware pipeline (PSR-15)
- Configuration management with environment variable support
- Database abstraction with support for MySQL, PostgreSQL, SQLite, and SQL Server
- ORM with Active Record pattern
- Database migrations
- Template engine with layout inheritance
- JWT authentication middleware
- CAPTCHA generation
- Flash messages
- Input validation
- Multiple cache backends (File, Memory, Redis, Predis, Memcache)
- Flexible logging with multiple backends (File, Daily, Stdio)
- Job queue system with Redis and ElasticMQ/SQS support
- Email sending with SMTP, Mailgun, and Resend transports
- Session management
- Event system
- Console commands and code generators
- Centralized error handling with JSON/HTML responses
- Development server command
